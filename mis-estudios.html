Asignatura: Desarrollo Full Stack

Tecnológico Nacional de México | Ingeniería Informática

Clave: DSH-2502

Descripción Académica

Esta asignatura capacita en el diseño, implementación y gestión de aplicaciones web completas, abarcando desde la arquitectura cliente-servidor hasta el despliegue continuo[cite: 12]. Se enfoca en crear soluciones modernas, accesibles y responsivas[cite: 14].

Competencias Desarrolladas

  • Diseño de interfaces web usables con HTML5, CSS3, Bootstrap y Angular[cite: 25].
  • Implementación de patrones de diseño en lógica de back-end para arquitecturas escalables[cite: 25].
  • Gestión de servicios RESTful e integración con Firebase[cite: 12].
  • Control de versiones con Git y despliegue continuo (CI/CD)[cite: 14].

Temario Principal

1. Fundamentos

Arquitectura cliente-servidor, SDN en capas, Node.js y control de versiones con GitHub[cite: 36].

2. Interfaces Modernas

HTML5 semántico, CSS3 (Flexbox/Grid), principios de accesibilidad WCAG y frameworks como Tailwind[cite: 36, 49].

3. Back-End Avanzado

Aplicación de patrones creacionales (Singleton, Factory), estructurales y de comportamiento.

4. Frameworks (Angular)

Uso de Angular CLI, componentes, formularios reactivos y consumo de APIs con HttpClient.

Proyectos y Prácticas

A lo largo de la formación se realizaron prácticas clave como:

  • Creación de menús adaptables y mejora de accesibilidad con etiquetas ARIA[cite: 69, 70].
  • Desarrollo de servicios de autenticación y almacenamiento con Firebase[cite: 59].
  • Implementación de sistemas de notas usando patrones de diseño como Decorator[cite: 73].
  • Consumo de APIs públicas mediante métodos HTTP (GET, POST, PUT, DELETE)[cite: 75].
